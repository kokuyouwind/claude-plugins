# Debugging Process Plugin

Systematic debugging methodology for Claude Code.

## Overview

Provides systematic debugging methodology for thorough root cause analysis with evidence-based investigation using the 5 Whys technique.

## Skills

### Debugging Process

**When activated:** When investigating code, analyzing bugs, debugging issues, performing root cause analysis, or handling errors.

**What it does:** Provides methodology for transparent real-time investigation with file:line references and enforces "investigate WHY before fixing" philosophy using 5 Whys technique.

## Installation

```bash
/plugin install dev-guidelines-debugging@kokuyouwind-plugins
```
