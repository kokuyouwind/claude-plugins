# Design Alternatives Plugin

Structured design decision evaluation for Claude Code.

## Overview

Guides structured evaluation of design options with trade-off analysis for architecture and implementation decisions.

## Skills

### Design Alternatives

**When activated:** When making design decisions, choosing architecture patterns, selecting technologies, or evaluating implementation approaches.

**What it does:** Guides the process of proposing multiple alternatives (2-3 minimum) with clear advantages, disadvantages, reasoning, and future scenarios.

## Installation

```bash
/plugin install dev-guidelines-design@kokuyouwind-plugins
```
